# 🍽️ Food Management System

A web-based **Food Management System** built using **Python Flask** that helps users manage food inventory and track expiration dates. The application allows users to add, edit, and delete food items while keeping track of their expiry to reduce food waste.

---

## 🚀 Features

* Add food items with details
* Edit and update food information
* Delete food items from inventory
* Track food expiration dates
* Display food inventory in a simple dashboard
* Barcode generation for food items
* User-friendly interface

---

## ⚙️ Installation & Setup

### 1. Clone the Repository

```bash
git clone https://github.com/pranav11-072/Food-Management.git
```

### 2. Navigate to the Project Folder

```bash
cd Food-Management
```

### 3. Install Required Packages

```bash
pip install flask
```

### 4. Run the Application

```bash
python app.py
```

### 5. Open in Browser

```
http://127.0.0.1:5000
```
